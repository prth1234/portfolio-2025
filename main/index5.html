Design

←23.70m→→17.60m←

Auf Basis der Grundsteine bauen wir das Webdesign aller Seiten auf.

Das Herz des Designs. Farben, Typography, Buttons, Abstände & Icons werden festgelegt.

Mikro- oder Scroll-Animationen werden grob animiert, um eine Idee davon zu haben, wie es live aussehen wird.

Alle Seiten werden für Desktop & Mobil gestaltet.

Im Beispiel kann man sehen, wie viel Arbeit im Design einer Website stecken kann – der Weg dahin, Iterationen, verworfene Konzepte etc.

Build

32T24T48PI

Das Webdesign wird in Webflow umgesetzt.

Alle Seiten werden nach WCAG 2.1 Level AA (außer anders abgestimmt) in Webflow umgesetzt.

Inhalte wie Seiten, Texte oder Bilder lassen sich einfach im CMS ändern und veröffentlichen.

Bei der Entwicklung wird auf eine SEO freundliche Umsetzung geachtet, damit deine Seite gefunden werden kann.

Run

K2

Läuft bei dir.

Die Website wird auf den meistgenutzten Geräten getestet.

Das Webflow-Projekt wird in deinen Account übertragen. Du bekommst eine Einweisung. Damit kannst du alleine weiterarbeiten – oder ich begleite dich weiterhin, um dich bei zukünftigen Änderungen und anderen Themen zu unterstützen.

Redirects werden eingerichtet. Deine Seite wird Live genommen. Aber nicht an einem Freitag.
